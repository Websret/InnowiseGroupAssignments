Задача 4. Разработка приложения на фреймворке (Laravel / Symfony)Задача 4.1. Разработка каталога продукции
Используемые технологии:  PHP, OOP, Laravel / Symfony, MySQL, MVC, Composer, HTML, CSS, JS, GIT, PHPUnit
Использование Docker обязательно для данной и всех последующих задач
Создать каталог продукции. Продуктами могут быть телевизоры, ноутбуки, мобильные телефоны, холодильники и др. Каждый
продукт имеет название, описание, производителя, дату выпуска и стоимость (в BYN). Услуги будут представлять собой
гарантийное обслуживание, доставку, установку и настройку. Каждая услуга будет иметь сроки и стоимость.
Каталог должен включать административную и пользовательскую части. Административная часть позволяет добавлять, изменять,
удалять продукты и услуги. Пользовательская часть предоставляет возможность просмотреть список товаров, открыть карточку
товара с информацией и перечнем возможных услуг. Пользователь может выбрать перечень дополнительных услуг. При выборе
услуг в карточке должна отобразиться общая стоимость товара с услугами.  
Добавить Unit тесты к решению. Добавить функциональный тест для контроллера.
Требования к реализации: применять паттерн MVC, придерживаться принципов SOLID, минималистичный интерфейс
Темы на обсуждение с ментором (изучаемая теория)
архитектура и базовые понятия используемого фреймворка, SOLID
