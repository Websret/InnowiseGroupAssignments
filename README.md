Task 4.6. Continuous integration and continuous delivery (CI/CD)

Используемые технологии:  PHP, OOP, Laravel / Symfony, MySQL, MVC, Composer, Twig, HTML, CSS, JS, GIT

Необходимо настроить CI/CD использую Git, Github и AWS для автоматического развертывания приложения на сервере по пушу в
master (main) ветку.

Одним из шагов пайплайна должен быть запуск юнит тестов, развертывание должно происходить только после успешного
прогона.

Примечание: Если в задаче 1 устанавливался LocalStack, то вместо AWS используем его.

Темы на обсуждение с ментором (изучаемая теория)

CI/CD, AWS, Web-hooks

Что считается успешным выполнением задания

полноценно работающее приложение

выполненное задание

использование всех указанных технологий

Срок выполнения задания:  4 рабочих дня
